# MVP 需求 - AI 照片编辑器（Web 版）

## 产品愿景

**面向内容创作者的 AI 驱动 Web 照片编辑工具**

核心差异化：**AI 智能裁剪和构图分析** - 上传照片，获得 AI 分析的最佳取景/裁剪建议，一键批量导出适配各平台尺寸。

**无需下载安装，浏览器即开即用，所有处理本地完成。**

---

## 市场定位

### 目标用户

**主要：内容创作者 / 博主**

- **年龄**：20-40 岁
- **平台**：桌面浏览器（Windows/macOS/Linux）
- **场景**：
  - 写博客需要配图
  - 经营多个社交媒体账号
  - 偶尔修图，不值得购买专业软件
  - 工作电脑无法安装软件

**次要：桌面端摄影爱好者**

- 偶尔处理照片
- 需要简单的裁剪和调整
- 看重隐私（不上传云端）

### 用户痛点

| 痛点 | 现有方案问题 | 我们的价值 |
|------|-------------|-----------|
| 每个平台尺寸不同 | 手动裁剪多次，麻烦 | 一键导出多尺寸 |
| 专业软件太重 | Lightroom/PS 体积大、贵、学习曲线陡 | 浏览器即用，简单直观 |
| 工作电脑受限 | 无法安装软件 | 纯 Web，无安装 |
| 担心隐私 | 不想上传照片到不知名网站 | 本地处理，照片不上传 |
| 偶尔使用 | 不值得付费购买 | 免费/低价，按需使用 |

## 产品需求（MVP）

### P0 核心需求

| ID | 需求 | 优先级 |
|----|------|--------|
| **R1** | **拖放上传** - 支持单张和多张 | P0 |
| **R2** | **AI 自动裁剪** - 检测主体，建议最佳构图 | P0 |
| **R3** | **多尺寸预览** - 并排显示不同裁剪选项 | P0 |
| **R4** | **一键应用** - 单击应用 AI 建议 | P0 |
| **R5** | **手动调整** - 裁剪框可拖动、调整 | P0 |
| **R6** | **基础调整** - 亮度、对比度、饱和度 | P0 |
| **R7** | **本地处理** - 无需上传，WebGL 加速 | P0 |
| **R8** | **多格式导出** - JPG/PNG/WEBP，可调质量 | P0 |
| **R9** | **批量导出** - 一次导出多个尺寸 | P0 |
| **R10** | **撤销/重做** - 非破坏性编辑 | P1 |

### 反向需求（不做）

| 不做什么 | 原因 |
|----------|------|
| 用户账号系统 | 增加使用门槛，隐私顾虑 |
| 云端存储 | 隐私问题，成本高 |
| 社交分享 | 浏览器限制，用户自己会分享 |
| 滤镜预设 | 非核心，AI 裁剪才是差异化 |
| 图层/蒙版 | 过于复杂，偏离目标用户 |

## 技术架构

详见 [技术架构文档](./architecture.md)

### 技术栈

```
前端框架：Vue 3 + TypeScript + Vite
图形渲染：Rust WASM + wgpu (WebGPU)
  • 所有图像处理在 GPU 上完成
  • 计算着色器加速批量操作
  • 零拷贝裁剪与纹理视图
AI/ML：ONNX Runtime WebGPU + YOLOv8-nano
  • GPU 加速推理
  • 本地运行，无数据上传
图像处理：wgpu WGSL 着色器 + image crate
UI 叠加：Canvas 2D（裁剪框、网格、手柄）
打包工具：Vite (前端) + wasm-pack (WASM)
部署：Vercel / Cloudflare Pages
```

### 架构图

```
┌─────────────────────────────────────────────┐
│         Vue3 前端层 (UI + 状态管理)          │
└─────────────────────────────────────────────┘
                    ↕ wasm-bindgen
┌─────────────────────────────────────────────┐
│    Rust WASM 层 (图像处理 + wgpu 渲染)       │
│  • 纹理管理、着色器、计算着色器              │
│  • 所有操作 GPU 加速                         │
└─────────────────────────────────────────────┘
                    ↕
┌─────────────────────────────────────────────┐
│   AI 推理层 (ONNX Runtime WebGPU)           │
│  • YOLOv8-nano GPU 加速推理                 │
└─────────────────────────────────────────────┘
```

### 免费版

```
├── 无限照片数量
├── AI 裁剪建议
├── 基础调整（亮度、对比度、饱和度）
├── 导出 JPG，最高 1080p
└── 无水印
```

### 专业版 - 一次性购买

```
$29.99（早鸟价 $19.99）

├── 所有免费版功能
├── 全分辨率导出
├── 批量导出多尺寸
├── WEBP/PNG 格式
├── 高级调整（可选）
└── 终身使用，无订阅
```

### 为什么不用订阅

- 目标用户是偶尔使用，订阅阻力大
- 竞品（Lightroom）都是订阅，差异化定价
- 一次性购买决策成本低
- Web 应用维护成本相对较低

## 开发路线图

### Phase 1：MVP（6-8 周）

- Week 1-2：项目搭建，基础 UI 框架
- Week 3-4：图片上传/显示，WebGL 渲染
- Week 5-6：AI 集成（YOLO + ONNX），裁剪建议
- Week 7-8：手动裁剪，导出功能

### Phase 2：完善（4-6 周）

- 调整面板（亮度、对比度、饱和度）
- 批量导出多尺寸
